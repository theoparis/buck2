Expand description

The computation Key that maps to a value. The key will be used as an index for caching the computed values.

ยงCloning

The Key will need to be cloned. This occurs whenever the key is recomputed and needs to be stored. During normal lookups, only a reference is needed.

The result also needs to be cloned whenever it is requested from DICE. Cloning therefore should be cheap.

fn validity(_x: &Self::Value) -> bool

If the computed value is false, DICE will consider that result to be a transient value that wonโt be re-used on subsequent computations. It will, however, reuse that value for all on-going computations at the current version.

The default here is true, but computations should override this if its expected that they may occasionally produce transient values.
