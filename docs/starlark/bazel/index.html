This file provides an implementation of LspContext specifically aimed at the use in a Bazel project. You can invoke it by using starlark --lsp --bazel. Note that only --lsp mode is supported.

This module is temporary, for the purpose of rapid iteration while the LSP interface develops. After the API of the LspContext trait stabilizes, this module will be removed, and extracted to its own project.
