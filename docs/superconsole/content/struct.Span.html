Expand description

A Span is a segment of text that may or may not have style applied to it. One may think of this as a unit of text. It must be valid to be constructed.

pub fn valid(stringlike: &str) -> bool

Spans are only valid if they do not contain any non whitespace characters other than a space. This is because the superconsole expects content to be mono-spaced and newlines, tabs, etc., violate this.

pub fn new_styled_raw(span: StyledContent<String>) -> Self

Create a Span that allows all whitespace characters, specially for emit/emit_aux scenarios.

Note: This method bypass whitespace validation, and should not be used for layout components. It’s specially designed for emit/emit_aux where content is displayed above the canvas and doesn’t require percise characters alignment.

pub fn iter(&self) -> impl Iterator<Item = Span> + '_

Iterates over each Grapheme in the Span. Applies the stylization of the Span to each Grapheme. Because a Grapheme is represented as another string, the sub-Span is represented as a Span. This panics if it encounters unicode that it doesn’t know how to deal with.
