Expand description

Generalized non-null pointer.

Anything which can be stored in LockFreeRawTable.

Required Associated Typesยง

Source

type Raw: Copy

The value stored in the table.

If the value is larger than max support atomic width on the platform, atomic operations will be spinlocked.

Source

type Ref<'a>: Copy where Self: 'a

Dereferenced value.
