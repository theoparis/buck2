Expand description

Spawn a future and return a DropcancelJoinHandle. The future will begin execution even before the handle is polled. The future will be able to observe and control its cancellation with the provided CancellationContext.

When the handle is dropped, the task will be cancelled. The task can be detached by calling DropcancelJoinHandle::detach.
